# PFMS App (Flutter + Spring Boot + Firebase)

PFMS = Personal Finance Management System  
- Frontend: Flutter (Web / Android)
- Backend: Spring Boot (Java 17) + Firebase Admin SDK + Firestore
- Auth: Firebase Authentication (Anonymous)
